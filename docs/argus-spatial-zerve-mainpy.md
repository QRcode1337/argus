# Argus Spatial Intelligence — Zerve `main.py`

Use this simplified FastAPI template for the Zerve spatial deployment.

## Important

- **Standalone Zerve deployment:** leave `APP_BASE_PATH` unset or empty.
- **Argus proxy mount at `/spatial`:** set `APP_BASE_PATH=/spatial`.

This version is prefix-aware, but defaults to root so it will work cleanly in Zerve without needing a mount path.

## Zerve settings

- **Deployment type:** Custom
- **Executor image:** FastAPI
- **Run command:**

```bash
uvicorn main:app --host 0.0.0.0 --port 8080
```

## Artifact layout

Put the generated spatial artifacts in a folder named `spatial/` next to `main.py`:

- `spatial/spatial_dashboard.html`
- `spatial/spatial_quality_scorecard.json`
- `spatial/postgis_extension_report.md`
- `spatial/spatial_index_audit.md`

## `main.py`

```python
from pathlib import Path
import os

from fastapi import FastAPI, HTTPException, Request
from fastapi.responses import HTMLResponse, JSONResponse, FileResponse

app = FastAPI(title="Argus Spatial Intelligence")

BASE_DIR = Path(os.environ.get("SPATIAL_BASE_DIR", ".")).resolve()
SPATIAL_DIR = BASE_DIR / "spatial"
APP_BASE_PATH = os.environ.get("APP_BASE_PATH", "").rstrip("/")


def bp(path: str = "") -> str:
    base = APP_BASE_PATH
    if not base:
        return path or ""
    if not path:
        return base
    return f"{base}{path}"


def exists(name: str) -> bool:
    return (SPATIAL_DIR / name).exists()


def read_text(name: str) -> str:
    path = SPATIAL_DIR / name
    if not path.exists():
        raise HTTPException(status_code=404, detail=f"Missing file: {name}")
    return path.read_text(encoding="utf-8")


def shell(title: str, body: str) -> str:
    return f"""<!doctype html>
<html>
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>{title}</title>
  <style>
    :root {{
      color-scheme: dark;
      --bg:#0f1115; --panel:#171a21; --panel2:#1f2430;
      --text:#e6e6e6; --muted:#9aa4b2; --accent:#7aa2f7; --border:#2a2f3a;
    }}
    body {{ margin:0; background:var(--bg); color:var(--text);
      font-family: Inter, system-ui, sans-serif; }}
    .wrap {{ max-width:1200px; margin:0 auto; padding:24px; }}
    .nav {{ display:flex; justify-content:space-between; flex-wrap:wrap; gap:12px; margin-bottom:18px; }}
    .brand {{ font-weight:800; }}
    a {{ color:var(--accent); text-decoration:none; }}
    .links {{ display:flex; gap:12px; flex-wrap:wrap; }}
    .links a {{ border:1px solid var(--border); padding:8px 12px; border-radius:10px; background:rgba(255,255,255,.02); }}
    .hero, .card {{ background:linear-gradient(180deg,var(--panel),var(--panel2)); border:1px solid var(--border); border-radius:16px; }}
    .hero {{ padding:24px; }}
    .title {{ margin:0 0 8px; font-size:2rem; }}
    .subtitle {{ color:var(--muted); margin:0; line-height:1.6; }}
    .btns {{ display:flex; flex-wrap:wrap; gap:10px; margin-top:18px; }}
    .btn {{ display:inline-block; padding:10px 14px; border-radius:10px; border:1px solid var(--border); color:var(--text); }}
    .btn.primary {{ background:var(--accent); color:#08101f; border-color:transparent; font-weight:700; }}
    .grid {{ display:grid; grid-template-columns:repeat(auto-fit,minmax(220px,1fr)); gap:12px; margin-top:16px; }}
    .card {{ padding:14px; }}
    .card h3 {{ margin:0 0 6px; font-size:1rem; }}
    .card p {{ margin:0; color:var(--muted); }}
    .content {{ padding:22px; line-height:1.6; }}
    pre {{ white-space:pre-wrap; overflow:auto; background:#0b0d12; padding:14px; border-radius:10px; border:1px solid var(--border); }}
    iframe {{ width:100%; height:82vh; border:1px solid var(--border); border-radius:16px; background:#111; }}
    .muted {{ color:var(--muted); }}
    .ok {{ color:#9ece6a; font-weight:700; }}
  </style>
</head>
<body>
  <div class="wrap">
    <div class="nav">
      <div class="brand">Argus Spatial Intelligence</div>
      <div class="links">
        <a href="{bp('/')}">Home</a>
        <a href="{bp('/dashboard')}">Dashboard</a>
        <a href="{bp('/scorecard')}">Scorecard</a>
        <a href="{bp('/report/postgis')}">PostGIS</a>
        <a href="{bp('/report/indexes')}">Indexes</a>
      </div>
    </div>
    {body}
  </div>
</body>
</html>"""


@app.get("/", response_class=HTMLResponse)
def home():
    body = f"""
    <div class="hero">
      <h1 class="title">Argus Spatial Intelligence</h1>
      <p class="subtitle">
        Spatial audit and visualization workspace for PostGIS outputs generated by Zerve.
        {'Mounted at <code>' + APP_BASE_PATH + '</code>.' if APP_BASE_PATH else 'Running at the Zerve root.'}
      </p>
      <div class="btns">
        <a class="btn primary" href="{bp('/dashboard')}">Open Dashboard</a>
        <a class="btn" href="{bp('/scorecard')}">View Scorecard</a>
        <a class="btn" href="{bp('/report/postgis')}">PostGIS Report</a>
        <a class="btn" href="{bp('/report/indexes')}">Index Audit</a>
      </div>

      <div class="grid">
        <div class="card"><h3>Dashboard</h3><p>{"Available" if exists("spatial_dashboard.html") else "Missing"}</p></div>
        <div class="card"><h3>Scorecard</h3><p>{"Available" if exists("spatial_quality_scorecard.json") else "Missing"}</p></div>
        <div class="card"><h3>PostGIS Report</h3><p>{"Available" if exists("postgis_extension_report.md") else "Missing"}</p></div>
        <div class="card"><h3>Index Audit</h3><p>{"Available" if exists("spatial_index_audit.md") else "Missing"}</p></div>
      </div>

      <p class="muted" style="margin-top:16px;">Service status: <span class="ok">ready</span> on port 8080</p>
      <p class="muted">Spatial dir: <code>{SPATIAL_DIR}</code></p>
      <p class="muted">Base path: <code>{APP_BASE_PATH or '/'}</code></p>
    </div>
    """
    return HTMLResponse(shell("Argus Spatial Intelligence", body))


@app.get("/dashboard")
def dashboard():
    path = SPATIAL_DIR / "spatial_dashboard.html"
    if not path.exists():
        raise HTTPException(status_code=404, detail="Missing spatial_dashboard.html")
    return FileResponse(path, media_type="text/html")


@app.get("/scorecard")
def scorecard():
    path = SPATIAL_DIR / "spatial_quality_scorecard.json"
    if not path.exists():
        raise HTTPException(status_code=404, detail="Missing spatial_quality_scorecard.json")
    return FileResponse(path, media_type="application/json")


@app.get("/report/postgis", response_class=HTMLResponse)
def postgis_report():
    text = read_text("postgis_extension_report.md")
    body = f"<div class='card content'><pre>{text}</pre></div>"
    return HTMLResponse(shell("PostGIS Report", body))


@app.get("/report/indexes", response_class=HTMLResponse)
def index_report():
    text = read_text("spatial_index_audit.md")
    body = f"<div class='card content'><pre>{text}</pre></div>"
    return HTMLResponse(shell("Spatial Index Audit", body))


@app.get("/api/status")
def api_status():
    return JSONResponse({
        "service": "argus-spatial-intelligence",
        "spatial_dir": str(SPATIAL_DIR),
        "base_path": APP_BASE_PATH or "/",
        "dashboard": exists("spatial_dashboard.html"),
        "scorecard": exists("spatial_quality_scorecard.json"),
        "postgis_report": exists("postgis_extension_report.md"),
        "index_report": exists("spatial_index_audit.md"),
    })
```

## Notes

- This version avoids optional dependencies like `markdown`.
- Leave `APP_BASE_PATH` unset for standalone Zerve deployment.
- Set `APP_BASE_PATH=/spatial` only when proxying through Argus under `/spatial`.
- If Zerve stores artifacts elsewhere, set:

```bash
SPATIAL_BASE_DIR=/path/to/artifacts
```

- This is the safest template to use if the fuller FastAPI version fails deployment.
